Analytics Page

The Analytics page allows draw up some reports on various statistics pertaining to the tests you have created, evaluated or taken. Listed below are the reports you can generate, along with a brief note about what the report is trying to tell you. The listing is grouped by role.


Tests as Creator

  1. Display scores of candidates in selected test: This report graphically displays the scores of different candidates in the selected test. This may be used by the test's creator (you) to identify the relative strength of the candidates on the topic of the selected test.
  2. Compare test popularity in selected topic: This report graphically displays the counts of candidates who took the various tests created by the user (you) under the selected topic. This may be used as a measure of the popularity of all tests created by you under the selected topic.
  3. Times at which selected test has been taken by candidates: This report graphically displays the date/time at which candidates have taken the selected test. A separate plot is displayed for each year.
  4. Max, Min, Mean, Median and Mode of scores of the selected test: This report graphically displays the mean value, median value and the mode value of the scores in the selected test.
  5. Chart to display test usages in selected topic: This report graphically displays the count of users who took the tests created by you under the selected topic. This may be used as a marker of popularity of the tests under the selected topic.
  6. Other tests taken by users who took the selected test: This report does a cohort analysis that displays the counts of users who took other tests created by you under the same topic, apart from taking the selected test.


Tests as Evaluator

  1. Ratio of number of pass grades to number of evaluated answers for a selected test: This displays 4 bars that reflect the total number of candidates for the selected test, the number of candidates who passed the selected test, the number of candidates who failed the test, and the number of candidates whose answer scripts have not been evaluated yet.
  2. Count of distinct tests evaluated under a selected topic: This displays the counts of all tests under the selected topic that have been evaluated by the user.
  3. Plot of answer scripts evaluated for a given topic over a span of time: This displays the plot of the counts of answer scripts that have been evaluated by the user (you) over time, for the selected topic. For example, if it shows a value of 16 on 9th May 2023, then you have evaluated 16 answer scripts of one or more tests under the selected topic on that date. This may be used as a measure of workload you have as an evaluator on the given date.


Tests as Candidate

  1. Compare with toppers: Compare your scores with the scores of toppers in all tests you have taken under the selected topic. If you have a very large number of tests under the selected topic, then it would be advisable to select specific tests under the topic to compare the score of that test with the score of the topper of that test.
  2. Compare with all: Compare your score with the scores of all other takers of the selected test. The names of the other users are replaced with the string 'hidden_x', where x is a number. This is to keep the identities of the users private.
  3. Compare your scores on a topic: Compare your scores for all tests you have taken on the selected topic. If a test has been attempted multiple times, then the score of only the last attempt is considered.
  4. Compare your scores by challenge with topper: Compare your scores on the various challenge/questions for the selected test with that of the topper in that test.
  5. Compute Mean, Median and Mode of all scores on a test: Compare your score in the selected test with the mean value, the median value, and the mode value of all takers of the test.
  6. Other tests taken by users who took the selected test: Display count of users of all tests that have been taken by the users who took the selected test. This is just a basic cohort analysis.
  7. Test scores under a selected topic against time: Compare your scores in the different tests you have taken under the selected topic. The analysis shows the scores on Y-axis and the time at which the tests were taken on the X-axis.
  8. Scores in percentage over time: Display the percentage scores of tests under the given topic as a function of time. The scores are on the Y-axis and the time in the X-axis.
  9. Performance on different challenge types: Comparison of your scores on different types of challenges in tests under the selected topic.
  10. Pass and Fail in tests under a selected topic against time: This is a plot of pass/fail in tests taken by you under the selected topic as a function of time.
