Search Page

The Search page allows you to search for tests and users using the name of the test or user, as well as using a search phrase or one or more keywords. Please note that this page doesn't allow you to search for a Group or Network on TestYard. You may search for Networks/Groups from the Network page.

You may also search for tests using the username of the creator of the test. However, since a user may be the owner/creator of large collection of tests, it would be beneficial to you to use the username as well as a search phrase to search.

For searching Network Groups, please refer to the help section for Network Page.
