Welcome to TestYard - Our Offerings

Parameterized Tests

We empower the user to create tests with a wide range of parameters. Users can create tests with multiple choice questions, subjective questions, programming challenges, and much more.

Secure Tests

Tests are conducted over secure connections and have the facility to recognize if the user leaves the test screen. Additionally, test question sequences are random. Check our documentation for more security features.

Instant Interviews

Using our interview interface, you can create instant audiovisual interviews. This will use your browser window on screen - you don't need to download/install any software for this. Easy.

Networks and Groups

Frequently, it so happens that we want to throw a test to a certain group of candidates. Think of a class test scenario. TestYard allows creation of groups of users for whom you can create a test schedule. Very convenient.

Monetize your Test Networks

You can create a premium group (called "Paid Group") from which you can earn when subscribers subscribe to it. Subscribers need to pay an amount (fixed by you, the owner of the group) when they join. More here.

Help at Hand

We are always available and willing to provide you with any technical support you need. If you want to have a dedicated system for your organization, we would be glad to set it up for you and get you started. This, however, is a premium service. See our subscription plans here.

Support for Scientific and Math Symbols.

TestYard tests support typing in math and science equations using symbols such as integrals, partial differentials, summations, greek alphabets, and much more. You can conduct and/or take a full-fledged university level mathematical test on it.

Basic Stats for Your Tests

TestYard provides you with some basic statistical information of all tests you have created, evaluated or taken.

Highly Active Development Team

The technical development team at TestYard is always looking to make your experience with the application smoother. To that effect, we are adding features and fixing bugs as and when we find them.
