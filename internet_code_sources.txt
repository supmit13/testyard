* make might throw an error pertaining to f95. Execute "make" like this: "make -j 1"
There error has got to do with available memory to execute - needs atleast 8 GB available space.
The above condition may or may NOT happen depending on the space you have on your system. If the space is sufficiently large, you possibly
won't receive this error. However, in case your space is limited and somewhere around 4GB (primary storage), you might face this issue.
**You have been warned.
In order to avoid this issue, please refer to the following discussion:
-----------------------------------------------------------------------
https://stackoverflow.com/questions/30887143/make-j-8-g-internal-compiler-error-killed-program-cc1plus

Process as discussed in the link above. Note - these commands need to be run as root user:
------------------------------------------------------------------------------------------
1. dd if=/dev/zero of=/swapfile bs=64M count=16
2. mkswap /swapfile
3. swapon /swapfile

To Clean them after use, please do this positively (DO NOT FORGET):
-------------------------------------------------------------------
1. swapoff /swapfile
2. rm /swapfile
